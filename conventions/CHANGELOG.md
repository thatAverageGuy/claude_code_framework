# Changelog Convention

## Format (Keep a Changelog)
## [Unreleased]
### Added
### Changed
### Fixed
### Removed

## [1.0.0] - YYYY-MM-DD
### Added
- T-001: Feature description

## Rules
- Update under [Unreleased] during every W2.
- Move [Unreleased] → versioned entry at release.
- One line per task. Reference TASK-ID.
- Never edit past versioned entries.
